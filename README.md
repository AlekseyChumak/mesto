# Проект: Место

## Проектная работа 4 для Яндекс Практикума.

### В проекте реализованы:
* Адаптивность для разных устройств
* Анимация при наведении курсора на ссылки
* Всплывающее окно с возможностью заполнения, реализовано в JS
* Блоки GRID
